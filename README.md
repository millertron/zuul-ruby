# zuul-ruby
Ruby port of Maze of Zuul, a text-based game (introductory programming assignment project at University, originally written in Java)

## Ruby version: 

2.3.1

## Instructions

This game only uses standard libraries, with no additional gems, so as long as you have Ruby (v2.3), you should be able to just download and play.
ZlGame is the main class, so to play the game, run the command:

```
ruby zl_game.rb
```
